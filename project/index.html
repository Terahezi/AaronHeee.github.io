Recommender System

General & Sequential Recommender
This is a class project where I explore basic algorithms in RS.
  • Implemented Bayesian Personalized Ranking algorithm on a books reading dataset.
  • Implemented Latent Factor Model algorithm on a books rating dataset.
  • Implemented Factorized Personalized Markoc Chain on a Steam reviews dataset for sequential recommendation.
  • Implemented Translation-based Recommender on a Steam reviews dataset for sequential recommendation.
  • Implemented Self-attentive Deep Neural Networks using ideas from Transformer on Steam reviews, Amazon Beauty/Games, MovieLens dataset for sequential recommendation.

    Online Movie Recommender using Spark
  • Loaded and parsed data in CVS files from MovieLens database to Spark RDD and used MapReduce to process RDD to get train, dev and test set.
  • Implemented Collaborative Filtering Algorithm and recursive least squares using SparkMLib.
  • Used Flask to build a web API, used CherryPy to build a WSGI server and ran the service on a Spark cluster.

    Distributed System and Network

    TritonHTTP
    This work is to build a webserver that implements a subset of the HTTP/1.1 protocol which we call it TritonHTTP. The code is in C++.
  • Implemented a client and a server based on a stream-oriented TCP protocol.
  • Appended features like reusing TCP connection for request and reply, using HTTP format as request and response message, allowing for pipelined requests.
  • Implemented error codes and custom error page to deal with various types of errors.

    3D Free Form Deformation (In Chinese)
    Apply the algorithm about 3D Free Form Deformation (FFD) with the toolkit VTK. Plus, utilizing the 3D Face Resruction model PRNet, we realize the workflow of 2D-Face -> 3D-Face -> 3D-FFD.     Natural Language Processing

    Intelligent Agent: Pac Man Search
    Adapted from the Berkeley Pac-Man Assignments originally created by John DeNero and Dan Klein. This is an AI course project to implement an agent to undertake specifical tasks, as well as to find the path to the end in mazes. What we do are: 1) implement path-search algorithms for DFS, BFS, UCS as well as A*, and 2) design effective heuristics. The report is here.
