Visual Question Answering
An Adversarial Learning Framwork with Knowledge Distillation from multiple 'teacher' networks. This is a novel and efficient technique about ensemble method (time cost is not more than a single model). That's a research project in Fudan and UIUC, collaborating with Dr. Zhiqiang Shen. This work will appear on AAAI 2019.   [PDF], [CODE] is coming soon.
Neural Attentive Item Similarity Model for Recommendation
A neural network model with attention mechanism for item-based Collaborative Filtering Model. This model has outperformed the art-of-the-state results. That's a research project in LMS Lab, led by Dr. Xiangnan He. This work has been published on TKDE 2018.

Adversarial Personalized Ranking for Recommendation
A new optimization framework, namely Adversarial Personalized Ranking (APR), which add adversarial noise when training, to enhance the robustness of a recommender model and thus improve its generalization performance. That's a research project in LMS Lab, led by Dr. Xiangnan He. This work has been published on SIGIR 2018.

Generative Discriminative Persona Classification Model
A LSTM model for Persona Classification task based on dialogue context. We design Traditional Machine Learning Model as baseline and three main models in our project based on seq2seq, which are Discriminative, Generative and Hierarchical Model. That's a research project with Shihan Ran and Xisen Jin in Fudan.
